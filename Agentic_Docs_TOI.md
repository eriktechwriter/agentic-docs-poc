## What is Agentic Docs?

AI-native documentation infrastructure that:

- Generates personalized content based on user role and context
- Assists writers with AI-powered authoring tools
- Integrates seamlessly with development workflows
- Produces targeted guides automatically

**Core principle:** Human expertise + AI augmentation = Superior outcomes

## Architecture Overview

Three-layer approach:

### Content Layer
Markdown/MDX with rich metadata

### Intelligence Layer
Vector database + AI processing

### Delivery Layer
Vision One docs panel + Doc site (behind Vision One login)

## Technology Components

### Content Format:
- Markdown for standard docs
- MDX for dynamic pages
- YAML frontmatter metadata
- Docusaurus for site

### AI Infrastructure:
- Vector database for semantic search

- Embedding models

October 21, 2024—The Detected Vulnerabilities widget in Exposure Overview now displays CVEs by impact level, including detected low-impact CVEs. New widgets in Threat and Exposure Management allow you to filter CVEs by high, medium, and low impact. To learn more about how CVE impact scores are calculated, see [CVE impact score](cve-impact-score.md).

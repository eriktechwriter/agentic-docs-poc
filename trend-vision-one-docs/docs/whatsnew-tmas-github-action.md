September 17, 2025—Use the new TMAS GitHub action to install TMAS for Code Security and Container Security deployments in Trend Vision One. This GitHub action uses automated CI/CD integration to run scans in every pull request or push to enforce policies directly in the pipeline.

Learn more about the TMAS GitHub action in the [GitHub README](https://github.com/trendmicro/tmas-scan-action/blob/main/README.md) or in [Integrate Trend Micro Artifact Scanner (TMAS) into a CI/CD pipeline](integrating-tmas-ci-cd-pipeline.md).

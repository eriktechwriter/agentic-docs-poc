title: Execution results
---

View the results each time an action runs on your automated tagging rules or mappings.

**Execution results** displays the status of the actions related to automated tagging rules or mappings, including when rules assign tags to assets, mappings sync tag values into Trend Vision One, and rules or mappings delete.

The following information displays for each action:

- **Execution ID**: The unique identifier of the action, with an icon indicating the status or result

- **Target**: The target of the action, including the following options:

  - Automated tagging rule

  - Cloud mapping

  - Microsoft Entra ID mapping

  - Active Directory mapping

- **Action**: The action taken on the target, including the following options:

  - Run rule

  - Sync mapping

  - Delete rule

  - Delete mapping

- **Name**: The name of the rule or mapping

- **Modified assets**: The number of assets affected when the rule or mapping ran

  :::tip
  Click the number of assets to view the **Execution details** for the action, including the specific tags assigned to or removed from assets.
  :::

- **Asset type**: The type of asset that the rule or mapping targets

- **Created**: The time at which the action entered the queue

- **Started**: The time at which the action began

- **Finished**: The time at which the action completed

title: Add cloud accounts

::: {.abstract}
You can add [AWS accounts](aws-accounts.md), [Azure subscriptions](azure-subscriptions.md), [Alibaba Cloud accounts](alibaba-cloud-accounts.md), [Google Cloud projects](file:/source/en-us/topics/google-cloud-projects.xml) and [Oracle Cloud Infrastructure (OCI) compartments](oracle-cloud-compartments.md)using Trend Vision One Cloud Risk Management.

To add a Google Cloud project, AWS account, Alibaba Cloud account, or an Azure subscription in Trend Vision One Cloud Risk Management:

1.  From *Cyber Risk Exposure Management* and *Misconfiguration and Compliance*

2.  Click on the *Add an Account* drop down.

3.  Select either **AWS Account**, **Azure Subscription**, **Alibaba Cloud Account,** **Google Cloud Project** or **Oracle Cloud** compartment.

4.  Follow the prompts on the *Cloud Account* app to complete adding your selected account provider. For detailed instructions, see [Get started with Cloud Accounts](getting-started-cloud-accounts.md).

For more info on connecting Cloud Accounts in Trend Vision One, see [Getting Started with Cloud Accounts](https://docs.trendmicro.com/en-us/enterprise/trend-vision-one/pointproductconnecti/cloud-account-manage/cam-getting-started.aspx)
:::

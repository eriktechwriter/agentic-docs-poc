title: Oracle Cloud support in Container Security
---

October 22, 2025—Trend Vision One now supports Oracle Cloud OKE in Container Security. Add an OKE cluster in **Inventory/Overview → Kubernetes → Oracle Cloud OKE** to add OKE clusters and view node and pod information. Use the Map to Cloud Account function to enable Cyber Risk Exposure Management.

For more information, see [Connect Oracle Cloud OKE clusters](connect-oracle-oke-clusters.md).

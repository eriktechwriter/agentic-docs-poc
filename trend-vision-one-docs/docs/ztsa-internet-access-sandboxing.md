title: Zero Trust Secure Access Internet Access supports sandboxing integration
---

Zero Trust Secure Access Internet Access now supports sandbox integration as part of a public preview, allowing you to automatically submit suspicious files to the Sandbox Analysis app.

:::note
You must set a daily reserve of more than zero to enable the automatic submission of suspicious files to the Sandbox Analysis app.

For instructions on setting a daily reserve, see [Submission Settings Configuration](submission-settings.md).
:::

For more information, see [Adding a threat protection rule](adding-threat-protection-rule.md).

**Zero Trust Secure Access → Secure Access Configuration → Internet Access and AI Service Access Configuration**

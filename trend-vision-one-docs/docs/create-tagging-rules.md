---

Create automated tagging rules to automatically assign tags to your assets.

### Procedure {#procedure}

1.  Go to **Tag Management â†’ Automated tagging**.

2.  Click **+ Create rule**.

    The **Create a tagging rule** window appears.

3.  Type a **Rule name**.

    :::note
    Rule names have a maximum length of 64 characters, and cannot have initial or final spaces. Rule names only support letters, numbers, spaces, and the following special characters: .@:+=\_/-
    :::

4.  Select an **Asset type** from the list for the rule to target.

    :::note
    You cannot update the asset type after you create the rule.
    :::

5.  Specify up to five **Conditions** for the tagging rule.

    1.  Choose an attribute for the condition.

    2.  Choose an operator.

    3.  Choose the other attribute for the condition.

    4.  Click **+ Condition** to add another condition.

6.  Use the toggle to enable or disable the rule.

    :::note
    Enabled rules run when you click **Save**, and every time you manually run the rule. You cannot run disabled rules.
    :::

7.  Select tags to assign to assets that meet the rule conditions.

    1.  Click **Custom tags** or **Platform tags**.

    2.  Select or search for tag values.

        :::note
        You can select up to 20 custom tag values for a rule to assign.
        :::

8.  Click **Save**.

    The new rule runs immediately, and appears in **Automated tagging**.

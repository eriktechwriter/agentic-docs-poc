July 1, 2024—Google Cloud Identity integration with Trend Vision One is now officially available. Seamlessly integrate with Google Cloud Identity to enhance your security visibility and response with streamlined access, management, and risk. Go to **Workflow and Automation → Third-Party Integrations** to set up the integration.

For more information, see [Google Cloud Identity integration](identity-integration-gcp.md).

title: Monitor and Scanning Network Traffic
---

Use additional Network Security features to monitor and scan network traffic passing through your connected network security appliances.

:::note
When you connect your Deep Discovery Inspector appliances to **Network Inventory** through Deep Discovery Director, **Monitoring / Scanning** configurations are not available. Go to the Deep Discovery Director management console to configure the corresponding settings.
:::

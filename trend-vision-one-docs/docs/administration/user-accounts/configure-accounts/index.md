Create and manage Trend Vision One user accounts to allow authorized users to access the console.

Before creating a SAML account, add your identity provider in the [Identity Providers](../../identity-providers) app.

**Related information**

- [Adding a SAML Group Account](./adding-a-saml-group-account "Invite all users in a group of your corporate identity provider (IdP) solution to sign in to the Trend Vision One console using their identities in the IdP solution.")
- [Add an IdP-Only SAML Group Account](./adding-idp-only-saml-group-account "Add multiple users in an assigned group to sign in to the Trend Vision One console using a corporate identity provider (IdP) solution.")
- [Add a local account](./adding-a-local-account "Add local accounts to allow users to sign in to the Trend Vision One console with their email address and password.")

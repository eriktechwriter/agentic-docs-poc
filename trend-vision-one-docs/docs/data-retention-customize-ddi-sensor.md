title: Data retention periods now customizable with credit allocation for Network Sensor and Deep Discovery Inspector
---

October 1, 2025—You can now customize data retention periods for Network Sensor and Trend Micro Deep Discovery Inspector in Data Source and Log Management. Credits will be allocated based on the selected retention periods. This flexibility allows you to tailor retention to meet your analysis, correlation, and threat hunting needs.

If you already have a data retention license for Network Sensor, only the retention period included in your license is available.

View the types of data and information supported for collection from Linux endpoints.

The following categories contain descriptions of the types of evidence that the [Collect Evidence task](collect-evidence-task.md) and [Trend Micro Incident Response Toolkit](manual-evidence-collection.md) collect from Linux endpoints. These evidence types appear in columns after selecting an evidence category when examining an [Evidence Report](evidence-report.md).

- [Basic Information](basic-information-linux.md)

- [Process Information](process-information-linux.md)

- [Service Information](service-information-linux.md)

- [Network Information](network-information-linux.md)

- [Account Information](account-information-linux.md)

- [User Activity](user-activity-linux.md)

- [Shared File Info Objects](shared-file-info-objects-linux.md)

:::note
Multiple evidence categories can include [Shared File Info Objects](shared-file-info-objects-linux.md).
:::

title: Using specified header matches criteria
---

Cloud Email Gateway Protection can scan the message headers for keyword expressions.

### Procedure {#procedure}

1.  On the **Scanning Criteria** tab, click **Advanced**.

2.  Select **Specified header matches**.

3.  Click the **keyword expressions** link.

4.  On the **Header Keyword Expressions** dialog, configure keywords.

    1.  Select or specify the message header fields whose value are used to match the defined keywords.

    2.  Select one or multiple available keyword expressions from the list, and click **Add \>**.

        To add keyword expressions that are not in the list, perform the following operations:

        1.  Click **Add**.

        2.  On the **Keywords and Expressions** screen, add new keyword expressions or edit existing keyword expressions as necessary. For details, see [Keyword expressions](keyword-expressions.md).

        3.  Click **Back to Policy**, and then open the **Header Keyword Expressions** dialog on the **Scanning Criteria** tab again.

        4.  Confirm the message header field settings, and then select the newly added keyword expressions from the list.

    3.  Click **Save**.

Trend Vision One Endpoint Security provides centralized and comprehensive Endpoint Security with a streamlined, single console experience â€“ a single place to deploy protection and policy, manage risk, and handle detection and response for your endpoints, servers, and workloads.

The steps below are a brief summary of how to set up Endpoint Security.

1.  Access your Trend Vision One console and make sure you have updated to the Foundation Services release.

    For more information, see [Getting started with Trend Vision One](../../getting-started/getting-started-with-trend-vision-one).

2.  Configure Endpoint Security and provision your endpoint protection solutions.

    - For new Trend Micro customers, see [Setting up Endpoint Security for new Trend Micro customers](./setup-endpoint-security-for-new).

    - For current Trend Micro customers, see [Update Trend Micro Endpoint Solutions](./update-trend-micro-endpoint-solutions).

      :::tip
      If you want to experience the new Trend Vision One Endpoint Security before updating all your endpoint protection solutions, see [Evaluate Trend Vision One Endpoint Security](./evaluate-trend-vision-one-endpoint-security).
      :::

3.  Configure your network environment

    [Deploy a Service Gateway](./deploy-a-service-gateway-and-configure-firewall-exceptions) to minimize [firewall configuration](./deploy-a-service-gateway-and-configure-firewall-exceptions) and streamline agent communication.

4.  [Manage your agent deployments](./manage-your-agent-deployments) by:

    - [Setting up endpoint groups](./manage-your-agent-deployments/manage-endpoint-groups)

    - [Creating default security policies](./manage-your-agent-deployments/create-default-endpoint-policies)

    - [Downloading the correct agent installation packages](./manage-your-agent-deployments)

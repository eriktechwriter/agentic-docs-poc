Endpoint Security provides a centralized location for managing detection and response on your endpoints. You can view detailed information in Endpoint Inventory, manage protection settings with Endpoint Security Policies, and deploy new agents.

- To get started with setting up your environment and deploying the Trend Vision One Endpoint Security agent, see [Getting started with Trend Vision One Endpoint Security](./getting-started-with-trend-vision-one-endpoint-security).

- To learn about reviewing and managing endpoints, see [Endpoint Inventory](./endpoint-inventory).

- To configure policies for your endpoints, see [Endpoint Security Configuration](./endpoint-security-configuration).

- To configure your Protection Managers and endpoint security settings, see:

  - [Standard Endpoint Protection](./standard-endpoint-protection)

  - [Server & Workload Protection](./server-workload-protection)

- To monitor the impact of agents on your security environment, see [Agent Resource Monitoring](./agent-resource-monitoring).

- To learn more about the endpoint interface, see [Trend Vision One Endpoint Security agent console](./trend-vision-one-endpoint-security-agent-console).

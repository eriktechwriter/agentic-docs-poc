title: Apply Intrusion Prevention best practices
---

The Intrusion Prevention Rule Compilation Failed error can occur due to a lack of resources on the machine, such as space, memory, or CPU. To help resolve this issue, apply the best practices on [Performance tips for intrusion prevention](../../../../configure-protection-modules/configure-intrusion-prevention/performance-tips-prev).

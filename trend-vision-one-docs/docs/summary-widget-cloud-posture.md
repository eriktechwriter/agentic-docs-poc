For a selected [Account](cloud-accounts-cloud-posture-setup.md), or [Group](grouped-accounts.md), the summary widget of the [Misconfiguration and Compliance](cloud-identity-dashboard-view.md) provides:

1.  [Compliance Score](compliance-score.md)
2.  [Check](model-check.md) summaries broken down successes, failures, and different severities

From the summary widget, you can action or access:

- Improve
- Browse all checks
- Run the Cloud Risk Management Bot
- Update tags
- Reports
- Settings

### Improve {#improve}

Report of all failed Checks for the account.

### Browse all checks {#browse-all-checks}

Access [All Checks Report](all-checks-report.md) which details all provides a list of all Check failures and successes for the account.

### Run the Compliance scan {#run-the-compliance-scan}

By default, the [Compliance scan](cloudIdentityBot.md) runs once every hour. You can run the Compliance scan on demand by clicking on "Run the Compliance scan" again.

### Update tags {#update-tags}

Organize your accounts into [Groups](grouped-accounts.md) using [Tags](cloud-account-tags.md). This functionality is available only when viewing an individually selected [Account](cloud-accounts-cloud-posture-setup.md).

### Reports {#reports}

Create, schedule, save and access [Configured Reports](configured-reports.md).

### Settings {#settings}

See:

- [account settings](cloud-acct-cloud-posture.md) when viewing a single [Account](cloud-accounts-cloud-posture-setup.md)
- [group settings](group-settings.md) when viewing a [Group](grouped-accounts.md)

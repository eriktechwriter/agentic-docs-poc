title: Manage compartment settings
---

Manage connected Oracle Cloud Infrastructure (OCI) compartment settings and update deployed resources.

Clicking an Oracle compartment name in Cloud Accounts opens the **Compartment Settings** screen. You can navigate between the following tabs:

<table>
<thead>
<tr>
<th><p>Tab</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Compartment information</p></td>
<td><p>View OCI compartment basic information and settings.</p>
<p>Use this tab to view and update the compartment name and description as displayed in the Cloud Accounts app.</p>
<p>For more information, see <a href="trend-vision-one-compartment-information">View and update compartment information</a>.</p></td>
</tr>
<tr>
<td><p>Resource update</p></td>
<td><p>Generate a terraform script to update the Trend Vision One resources deployed to your account.</p>
<p>The steps to update resources vary depending on the deployment method used:</p>
<ul>
<li><p>For Oracle Resource Manager deployments, see <a href="trend-vision-one-oracle-cloud-resource-update">Update resources using Oracle Resource Manager</a>.</p></li>
<li><p>For LocalShell deployments, see <a href="trend-vision-one-update-resources-using-localshell">Update resources using LocalShell</a>.</p></li>
</ul></td>
</tr>
</tbody>
</table>

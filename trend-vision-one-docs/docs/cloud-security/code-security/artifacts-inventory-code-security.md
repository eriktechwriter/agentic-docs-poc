From the Artifacts/Inventory page, you can view your Code Security scan findings, including artifact names, policy violation detections, and the last scan time.

Trend Micro Artifact Scanner (TMAS) detects open source vulnerabilities, malware, and secrets in the following artifact types:

- Container images
  - docker
  - podman
  - docker-archive
  - oci
  - singularity
  - registry
- Binary files
- Single files
- Directory paths
- Source code

See [Supported artifacts](../container-security/log/container-image-scanning/trend-micro-artifact-scanner-tmas/artifact-scanner-cli) in the TMAS documentation for more information.

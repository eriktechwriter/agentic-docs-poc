title: Compliance scan frequency
---

By default, the Compliance scan runs every hour, however, there is the option to modify the intervals between each bot run.

### Steps to modify Compliance scan frequency {#steps-to-modify-compliance-scan-frequency}

1.  Input a value for *Delay between runs*.
2.  Click *Update Scan settings.*
3.  Once modified, the interval will be displayed on the [Settings page](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/manage-cloud-accounts/account-settings/cloud-acct-cloud-posture). {.zoom}

:::note
Compliance scan does not allow you to set up a frequency of less than an hour.
:::

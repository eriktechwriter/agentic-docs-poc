Trend Vision One™ – Cloud Risk Management supports notifications sent via PagerDuty.

### Set up a PagerDuty channel {#set-up-a-pagerduty-channel}

1.  Click on *Create an PagerDuty channel* and
2.  [Set automatic notifications](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/communication-and-notification/communication-settings/toggle-automatic-notifications)
3.  [Configure Triggers](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/communication-and-notification/communication-settings/communication-triggers)
4.  *Configure services* requires the following two user inputs to set up a channel
    1.  *Service Name*
    2.  *Service Key:* the '*Integration Key'* for your service created in PagerDuty for Cloud Risk Management alerts
5.  [Copy communication settings](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/communication-and-notification/communication-settings/copy-communication-settings)

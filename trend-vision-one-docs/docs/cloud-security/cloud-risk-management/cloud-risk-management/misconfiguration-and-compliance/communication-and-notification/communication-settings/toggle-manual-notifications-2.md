title: Toggle manual notifications

The following channels have a manual notifications option

- [Jira Communication](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/communication-and-notification/communication-channels/jira-communication/jira-integration)
- [Zendesk Communication](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/communication-and-notification/communication-channels/zendesk-communication)
- [ServiceNow ITSM Communication](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/communication-and-notification/communication-channels/servicenow-communication)
- [Amazon SNS Communication](../../../../../../cyber-risk-exposure-management/security-posture-management/cloud-risk-management/misconfiguration-and-compliance/communication-and-notification/communication-channels/amazon-sns-communication)

By default, manual notifications are set to *OFF* for these channels.

If manual notifications is turned *ON*, you can send notifications for failures by navigating to specific rules on the Account dashboard.

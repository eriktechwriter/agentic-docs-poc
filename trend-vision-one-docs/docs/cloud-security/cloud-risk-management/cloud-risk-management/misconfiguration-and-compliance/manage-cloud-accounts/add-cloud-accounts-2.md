::: {.abstract}
You can add [AWS accounts](../../../../cloud-accounts/aws-accounts), [Azure subscriptions](../../../../cloud-accounts/azure-subscriptions), [Alibaba Cloud accounts](../../../../cloud-accounts/alibaba-cloud-accounts), [Google Cloud projects](file:/source/en-us/topics/google-cloud-projects.xml) and [Oracle Cloud Infrastructure (OCI) compartments](../../../../cloud-accounts/oracle-cloud-infrastructure-oci-compartments)using Trend Vision One Cloud Risk Management.

To add a Google Cloud project, AWS account, Alibaba Cloud account, or an Azure subscription in Trend Vision One Cloud Risk Management:

1.  From *Cyber Risk Exposure Management* and *Misconfiguration and Compliance*

2.  Click on the *Add an Account* drop down.

3.  Select either **AWS Account**, **Azure Subscription**, **Alibaba Cloud Account,** **Google Cloud Project** or **Oracle Cloud** compartment.

4.  Follow the prompts on the *Cloud Account* app to complete adding your selected account provider. For detailed instructions, see [Get started with Cloud Accounts](../../../../cloud-accounts/get-started-with-cloud-accounts).

For more info on connecting Cloud Accounts in Trend Vision One, see [Getting Started with Cloud Accounts](https://docs.trendmicro.com/en-us/enterprise/trend-vision-one/pointproductconnecti/cloud-account-manage/cam-getting-started.aspx)
:::

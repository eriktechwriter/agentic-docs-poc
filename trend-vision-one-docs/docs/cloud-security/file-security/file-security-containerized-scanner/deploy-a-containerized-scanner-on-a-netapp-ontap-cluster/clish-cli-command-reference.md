The Management Service CLI provides two main command categories:

- [Agent Management](./agent-management-commands): Commands for managing ONTAP agents, their configurations, and status monitoring

- [Scanner Management](./scanner-management-commands): Commands for configuring scanner policies and settings

Access the CLI using:

``` codeblock
kubectl exec -it deploy/scanner-visionone-filesecurity-management-service -- bash
# Then run: clish <command>
```

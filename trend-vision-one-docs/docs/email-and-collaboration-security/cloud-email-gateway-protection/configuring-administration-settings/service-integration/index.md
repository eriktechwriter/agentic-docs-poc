Currently, Cloud Email Gateway Protection integrates with the following Trend Micro products:

- Trend Micro Apex Central

  For more information about Trend Micro Apex Central, see [Apex Central](../../about-cloud-email-gateway-protection/integration-with-trend-micro-products/apex-central).

Furthermore, Cloud Email Gateway Protection supports API openness to allow integration with external systems via APIs. For example, Cloud Email Gateway Protection opens REST APIs to allow customers to query domains; query, add, replace, and delete directory users; and retrieve policy event logs and mail tracking logs for the purpose of third-party SIEM application integration.

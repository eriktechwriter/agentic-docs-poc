title: Using body matches criteria
---

Cloud Email Gateway Protection can scan the message body for keyword expressions.

### Procedure {#procedure}

1.  On the **Scanning Criteria** tab, click **Advanced**.

2.  Select **Body matches**.

3.  Click the "**keyword expressions**" link.

4.  Configure keywords.

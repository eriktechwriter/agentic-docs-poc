To go to the **Inbound Protection** screen, choose **Email and Collaboration Security Operations → Cloud Email Gateway Protection → Inbound Protection**.

To go to the **Outbound Protection** screen, choose **Email and Collaboration Security Operations → Cloud Email Gateway Protection → Outbound Protection**.

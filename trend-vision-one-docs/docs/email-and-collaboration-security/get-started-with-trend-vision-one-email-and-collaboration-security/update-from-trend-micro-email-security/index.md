title: Update from Trend Micro Email Security
---

Update your Trend Micro Email Security instance to Cloud Email Gateway Protection to enable the security features of Trend Vision One Email and Collaboration Security.

Before updating Trend Micro Email Security to the new Cloud Email Gateway Protection, you must sign up and complete the Trend Vision One registration process. If you are a current Trend Vision One customer, make sure to [update to the Foundation Services Release](../../../getting-started/getting-started-with-trend-vision-one/update-trend-vision-one-to-the-foundation-services-release) if you have not already done so.

Updating to Cloud Email Gateway Protection has some limitations. If your Trend Micro Email Security instance meets one of the following criteria, you might not be able to update.

- Your Trend Micro Email Security is managed by Trend Micro Remote Manager

- Your Trend Micro Email Security license expired and it has been more than 30 days since expiration

Once you are ready, connect and update your Trend Micro Email Security instance.

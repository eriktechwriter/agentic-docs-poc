title: Create custom tags
---

Add new custom tags to assign to assets and improve your asset organization.

### Procedure {#procedure}

1.  Go to **Tag Management → Tag library → All custom tags**.

2.  Click **+ Add tag**.

    The **Add tags** window appears.

3.  Type a **Tag property**.

    :::note
    Tag properties have a maximum length of 64 characters, and cannot have initial or final spaces. Tag property names only support letters, numbers, spaces, and the following special characters: .@:+=\_/-
    :::

4.  Type one or more **Tag values** for the tag property.

    :::note
    You can add up to 35 values for each tag property.

    Tag values have a maximum length of 64 characters, and cannot have initial or final spaces. Tag value names only support letters, numbers, spaces, and the following special characters: .@:+=\_/-
    :::

5.  If you want to create more new custom tags, you can click **+ Add tag** and repeat the previous steps.

6.  Click **Add**.

    The new custom tag property appears in **All custom tags**.

    You can expand the list of tag values and click **Manage assets** next to any tag value to [assign or remove the tag from assets](./managing-tagged-assets).

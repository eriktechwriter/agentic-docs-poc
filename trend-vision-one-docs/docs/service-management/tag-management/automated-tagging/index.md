Create, edit, and delete tagging rules to automatically tag assets that meet the rule conditions.

The following table outlines the actions available in **Automated tagging**.

<col style="width: 75%" />
</colgroup>

<tr>
<th><p>Action</p></th>
<th><p>Description</p></th>
</tr>
</thead>

<tr>
<td><p>Create automated tagging rules</p></td>
<td><p>Click <strong>+ Create rule</strong> to <a href="trend-vision-one-create-tagging-rules">add a new rule</a> for automated tagging.</p></td>
</tr>
<tr>
<td><p>Edit automated tagging rules</p></td>
<td><p>Click the <img src="./images/edit_icon=GUID-1F1D1164-5310-4D6D-ACD0-6049C86960AF.webp" /> icon to <a href="trend-vision-one-edit-tagging-rule">edit an existing rule</a>.</p></td>
</tr>
<tr>
<td><p>Run automated tagging rules</p></td>
<td><p>Click the <img src="./images/run_icon=cbe6ecd0-17e8-4e04-bef3-4efe4eb9c7e5.webp" /> icon to immediately run an enabled rule.</p>

:::tip

<p>To run a disabled rule, you must first edit the rule and change the status to <strong>Enabled</strong>.</p>

:::

</td>
</tr>
<tr>
<td><p>Delete automated tagging rules</p></td>
<td><p>Click the <img src="./images/trash_icon=GUID-47cf6867-6315-438e-8670-86ff36f22a28.webp" /> icon to delete a tagging rule.</p>

<p>Deleting a rule removes all tags previously assigned to assets by the rule.</p>

:::

</td>
</tr>

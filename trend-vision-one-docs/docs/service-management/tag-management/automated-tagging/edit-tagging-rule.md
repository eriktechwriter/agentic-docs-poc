title: Edit automated tagging rules
---

Update the basic details, rule conditions, or tag values assigned by an automated tagging rule.

### Procedure {#procedure}

1.  Click the ![](/images/edit_icon=GUID-1F1D1164-5310-4D6D-ACD0-6049C86960AF.webp) icon next to an automated tagging rule.

    The **Edit a tagging rule** window appears.

2.  Edit the name, conditions, status, or tag values assigned by the rule.

    :::note
    You cannot edit the **Asset type** targeted by an automated tagging rule.
    :::

3.  Click **Save**.

    The automated tagging rule updates.

    :::warning[Important]
    If one or more assets no longer meet the updated rule conditions, this action removes the tags previously assigned by the rule to those assets. The rule automatically assigns tags to any assets that meet the updated rule conditions.
    :::

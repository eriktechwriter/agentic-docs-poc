title: Component Updates
---

The Trend Micro Apex Central server hosts component files that the managed products use to keep your network protected from the latest security threats.

Keep the components up-to-date by running manual or scheduled updates. Trend Micro Apex Central allows you to perform the following tasks:

- Download the latest component versions from an update source

- Deploy updated components to managed products

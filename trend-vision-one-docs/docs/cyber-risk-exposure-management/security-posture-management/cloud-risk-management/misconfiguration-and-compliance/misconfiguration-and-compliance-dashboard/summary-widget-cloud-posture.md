For a selected [Account](../manage-cloud-accounts/cloud-accounts-cloud-posture-setup), or [Group](../manage-cloud-accounts/account-settings/manage-account-groups/grouped-accounts), the summary widget of the [Misconfiguration and Compliance](../cloud-identity-dashboard-view) provides:

1.  [Compliance Score](../compliance/compliance-score)
2.  [Check](../misconfiguration-and-compliance-rules/checks/model-check/model-check) summaries broken down successes, failures, and different severities

From the summary widget, you can action or access:

- Improve
- Browse all checks
- Run the Cloud Risk Management Bot
- Update tags
- Reports
- Settings

### Improve {#improve}

Report of all failed Checks for the account.

### Browse all checks {#browse-all-checks}

Access [All Checks Report](../reports/all-checks-report) which details all provides a list of all Check failures and successes for the account.

### Run the Compliance scan {#run-the-compliance-scan}

By default, the [Compliance scan](../compliance-scan-help/cloudIdentityBot) runs once every hour. You can run the Compliance scan on demand by clicking on "Run the Compliance scan" again.

### Update tags {#update-tags}

Organize your accounts into [Groups](../manage-cloud-accounts/account-settings/manage-account-groups/grouped-accounts) using [Tags](../manage-cloud-accounts/account-settings/manage-cloud-account-tags/cloud-account-tags). This functionality is available only when viewing an individually selected [Account](../manage-cloud-accounts/cloud-accounts-cloud-posture-setup).

### Reports {#reports}

Create, schedule, save and access [Configured Reports](../reports/configured-reports).

### Settings {#settings}

See:

- [account settings](../manage-cloud-accounts/account-settings/cloud-acct-cloud-posture) when viewing a single [Account](../manage-cloud-accounts/cloud-accounts-cloud-posture-setup)
- [group settings](../manage-cloud-accounts/account-settings/manage-account-groups/group-settings) when viewing a [Group](../manage-cloud-accounts/account-settings/manage-account-groups/grouped-accounts)

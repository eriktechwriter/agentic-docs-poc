title: Toggle manual notifications
---

The following channels have a manual notifications option

- [Jira Communication](../communication-channels/jira-communication/jira-integration)
- [Zendesk Communication](../communication-channels/zendesk-communication)
- [ServiceNow ITSM Communication](../communication-channels/servicenow-communication)
- [Amazon SNS Communication](../communication-channels/amazon-sns-communication)

By default, manual notifications are set to *OFF* for these channels.

If manual notifications is turned *ON*, you can send notifications for failures by navigating to specific rules on the Account dashboard.

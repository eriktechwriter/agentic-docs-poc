Learn how to create, delete, and configure workspaces.

Create and manage [workspaces](./workspaces), organize collected evidence with workspaces, and [create investigation timelines](./timeline) on the **War Room** tab.

The following table outlines the actions available on the **War Room** tab.

<table>
<colgroup>
<col style="width: 25%" />
<col style="width: 75%" />
</colgroup>
<thead>
<tr>
<th><p>Action</p></th>
<th><p>Description</p></th>
</tr>
</thead>
<tbody>
<tr>
<td><p>Create a workspace</p></td>
<td><p>Organize evidence, construct investigation timelines, and triage endpoints in your environment with workspaces.</p>
<p>Click <strong>Create Workspace</strong> to create a new workspace.</p></td>
</tr>
<tr>
<td><p>View a workspace</p></td>
<td><p>Click the workspace name to go to the workspace.</p></td>
</tr>
<tr>
<td><p>Search for a workspace by name</p></td>
<td><p>Use the <strong>Keyword</strong> search field to locate specific workspaces by name.</p></td>
</tr>
<tr>
<td><p>View a case</p></td>
<td><p>Keep track of incident investigations, procedures, and supplemental information for threat alerts and risk events with <a href="trend-vision-one-case-management">cases</a>.</p>
<p>If a Forensics workspace is part of a case in Case Management, click the case ID to display the case in the <a href="trend-vision-one-case-viewer">Case Viewer</a>.</p></td>
</tr>
<tr>
<td><p>Update impacted endpoints</p></td>
<td><p>In Case Viewer, click <strong>Update Forensics Workspace</strong> to update a workspace with impacted endpoints.</p>
<p>If the case no longer includes an endpoint, Trend Vision One does not automatically remove the endpoint. You can manually delete any unaffected endpoints from the workspace.</p></td>
</tr>
<tr>
<td><p>Additional actions</p></td>
<td><p>Locate a workspace, click <img src="./images/options=ddb0b67f-0654-4aa5-8bc7-48ec554c5448.webp" /> at the end of the row, and select any of the available actions:</p>
<ul>
<li><p><strong>Delete Workspace</strong> to delete a workspace.</p></li>
<li><p><strong>Modify Workspace</strong> to change the name or description of a workspace.</p></li>
</ul></td>
</tr>
</tbody>
</table>

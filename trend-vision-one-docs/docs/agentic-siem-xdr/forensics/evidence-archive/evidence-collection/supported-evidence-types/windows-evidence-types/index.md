title: Windows evidence types
---

View the types of data and information supported for collection from Windows endpoints.

The following categories describe the types of evidence the [Incident Response Evidence Collection playbooks](../../../../../../workflow-and-automation/security-playbooks/template-based-playbooks/create-incident-response-evidence-collection-playbooks), [Collect Evidence response task](../../../../../../workflow-and-automation/response-management/response-actions/collect-evidence-task), and [Trend Micro Incident Response Toolkit](../../manual-evidence-collection) collect from Windows endpoints.

These evidence types appear in columns after selecting an evidence category when examining an Evidence Report.

View the types of data and information supported for collection from Linux endpoints.

The following categories contain descriptions of the types of evidence that the [Collect Evidence task](../../../../../../workflow-and-automation/response-management/response-actions/collect-evidence-task) and [Trend Micro Incident Response Toolkit](../../manual-evidence-collection) collect from Linux endpoints. These evidence types appear in columns after selecting an evidence category when examining an [Evidence Report](../../../../war-room/evidence-report).

- [Basic Information](./basic-information-linux)

- [Process Information](./process-information-linux)

- [Service Information](./service-information-linux)

- [Network Information](./network-information-linux)

- [Account Information](./account-information-linux)

- [User Activity](./user-activity-linux)

- [Shared File Info Objects](./shared-file-info-objects-linux)

:::note
Multiple evidence categories can include [Shared File Info Objects](./shared-file-info-objects-linux).
:::
